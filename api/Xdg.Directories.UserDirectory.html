Remarks

For all user directories, the directories are resolved as follows:

  1. The respective XDG_* enivronment variable will always be used if specified
  2. Windows Use Known Folders.
    macOS Not sure where this is documented by Apple but it largely follows what Linux does.
    Linux/FreeBSD Follow what xdg-user-dirs sets.
